# ✨ نظام قص الصور المتقدم - الإصدار 3.0

## 🎯 الميزات الرئيسية

### 1. التحكم اليدوي بالسحب ✋
- **اسحب على الصورة** لتحديد المنطقة مباشرة
- **معاينة فورية** أثناء السحب
- **تحديث تلقائي** للقيم الرقمية

### 2. التحكم الرقمي الدقيق 🔢
- **حقول إدخال** للتحكم بالبكسل
- **موضع X و Y**: تحديد نقطة البداية
- **العرض والطول**: تحديد الأبعاد

### 3. إجراءات سريعة ⚡
- كامل الصورة
- النصف العلوي
- النصف السفلي
- المركز

## 🎓 أمثلة

### مثال 1: قص سؤال
```
1. افتح الصورة
2. اسحب من بداية السؤال لنهايته
```

```
1. افتح الصورة

```

### مثال 3: قص النصف العلوي
