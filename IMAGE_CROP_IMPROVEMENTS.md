- **معاينة فورية** أثناء السحب

### 2. **التحكم الرقمي الدقيق** 🔢
- **حقول إدخال رقمية** للتحكم الدقيق

### 3. **التكامل بين الطريقتين** 🔄

3. عدّل القيم الرقمية إذا لزم الأمر
4. احفظ!

## 🎨 التفاصيل التقنية

### كيف يعمل السحب؟

```dart
1. عند لمس الشاشة (onPanStart):
   - تسجيل نقطة البداية
   - تفعيل وضع السحب

2. أثناء السحب (onPanUpdate):
   - تحديث نقطة النهاية
   - حساب الإحداثيات على الصورة الفعلية
   - تحديث منطقة القص
   - تحديث القيم الرقمية

3. عند رفع الإصبع (onPanEnd):
   - إيقاف وضع السحب
   - تثبيت المنطقة المحددة
```

### التحويل من إحداثيات الشاشة إلى الصورة:

```dart
// حساب معامل التحجيم
scale = min(containerWidth / imageWidth, containerHeight / imageHeight)

// حساب الإزاحة (centering)
offsetX = (containerWidth - scaledImageWidth) / 2
offsetY = (containerHeight - scaledImageHeight) / 2

// تحويل الإحداثيات
imageX = (screenX - offsetX) / scale
imageY = (screenY - offsetY) / scale
```

## 📝 الكود الرئيسي

### معالجة السحب:

```dart
void _onPanStart(DragStartDetails details) {
  setState(() {
    _isDragging = true;
    _dragStart = details.localPosition;
    _dragEnd = details.localPosition;
  });
}

void _onPanUpdate(DragUpdateDetails details) {
  setState(() {
    _dragEnd = details.localPosition;
    _updateCropFromDrag();
  });
}

void _onPanEnd(DragEndDetails details) {
  setState(() {
    _isDragging = false;
    _updateCropFromDrag();
  });
}
```

### تحويل الإحداثيات:

```dart
void _updateCropFromDrag() {
  // حساب معامل التحجيم
  final scale = min(containerWidth / imageWidth, containerHeight / imageHeight);
  
  // حساب الإزاحة
  final offsetX = (containerWidth - scaledImageWidth) / 2;
  final offsetY = (containerHeight - scaledImageHeight) / 2;
  
  // تحويل إحداثيات الشاشة إلى إحداثيات الصورة
  final startX = ((dragStart.dx - offsetX) / scale).clamp(0, imageWidth);
  final startY = ((dragStart.dy - offsetY) / scale).clamp(0, imageHeight);
  
  // تحديث القيم
  _x = startX.toInt();
  _y = startY.toInt();
  // ... إلخ
}
```
