# دليل الاستخدام السريع - قص الصور v3.0

## 🎯 طريقتان للتحكم!

### 1️⃣ السحب اليدوي (الأسرع) ✋
```
المس → اسحب → ارفع → تم! ✅
```

### 2️⃣ الإدخال الرقمي (الأدق) 🔢
```
أدخل الأرقام → شاهد المعاينة → احفظ! ✅
```

---

## ⚡ البداية السريعة

### الطريقة الأولى: السحب المباشر
1. افتح شاشة القص
2. **المس** الصورة عند بداية المنطقة المطلوبة
3. **اسحب** إلى نهاية المنطقة
4. **ارفع** إصبعك
5. ✅ **تم!** - القيم تحدثت تلقائياً

### الطريقة الثانية: الأرقام
1. افتح شاشة القص
2. **أدخل** القيم في الحقول:
   - موضع X (من اليمين)
   - موضع Y (من الأعلى)
   - العرض
   - الطول
3. **شاهد** المعاينة تتحدث
4. ✅ **تم!**

### الطريقة الثالثة: المزج (الأفضل) 🌟
1. استخدم **الإجراءات السريعة** (مثلاً: النصف العلوي)
2. **اسحب** لتعديل المنطقة
3. **عدّل الأرقام** للدقة
4. ✅ **تم!**

---

## 💡 أمثلة سريعة

### مثال 1: قص سؤال من امتحان
**بالسحب**:
```
المس أعلى السؤال → اسحب لأسفل السؤال → ارفع → احفظ
```

**بالأرقام**:
```
X: 100, Y: 800, العرض: 1800, الطول: 400 → احفظ
```

### مثال 2: قص النصف العلوي
```
اضغط "النصف العلوي" → احفظ
```

### مثال 3: قص منطقة مخصصة
```
اضغط "المركز" → اسحب لتعديل → عدّل الأرقام → احفظ
```

## 🎨 فهم الشاشة

### ما تراه:
- **الصورة الأصلية**: في المنتصف
- **المنطقة الداكنة**: سيتم حذفها ❌
- **المنطقة الواضحة**: سيتم حفظها ✅
- **الإطار الأزرق**: حدود القص
- **الشبكة البيضاء**: للمحاذاة
- **الدوائر**: مقابض الزوايا

### الرسالة التوجيهية:
عندما ترى: **"اسحب على الصورة لتحديد المنطقة"**
- معناها: يمكنك السحب الآن ✋
- تختفي أثناء السحب
- تعود بعد رفع الإصبع

---

## 🔢 فهم الحقول

### موضع X (أفقي):
- `0` = أقصى اليمين ➡️
- `1000` = 1000 بكسل من اليمين
- كلما زاد → اتجه لليسار ⬅️

### موضع Y (عمودي):
- `0` = أعلى الصورة ⬆️
- `500` = 500 بكسل من الأعلى
- كلما زاد → اتجه للأسفل ⬇️

### العرض والطول:
- بالبكسل (px)
- يجب أن يكون > 0
- لا يتجاوز حجم الصورة

---

## 🎯 نصائح ذهبية

### ✅ افعل:
1. **ابدأ بالسحب** - أسرع طريقة
2. **استخدم الإجراءات السريعة** - للبداية
3. **عدّل بالأرقام** - للدقة
4. **راقب المعاينة** - دائماً
5. **جرب أكثر من طريقة** - اختر الأنسب

### ❌ لا تفعل:
1. لا تسحب خارج الصورة
2. لا تتجاهل المعاينة
3. لا تنسى الحفظ

---

## 🚀 الإجراءات السريعة

### كامل الصورة 🖼️
- يعيد تعيين كل شيء
- استخدمه للبداية من جديد

### النصف العلوي ⬆️
- يحدد النصف العلوي تلقائياً
- مفيد لقص الأسئلة العلوية

### النصف السفلي ⬇️
- يحدد النصف السفلي تلقائياً
- مفيد لقص الأسئلة السفلية

### المركز 🎯
- يحدد المنطقة المركزية (50%)
- مفيد للبداية ثم التعديل

## 📊 بطاقة المعلومات

### ما تعرضه:
```
حجم الصورة الأصلي: 2000 × 3000 بكسل
  ↑ معلومة عن الصورة الكاملة

حجم المنطقة المحددة: 1800 × 400 بكسل
  ↑ ما سيتم حفظه

نسبة القص: 12.0%
  ↑ نسبة المنطقة من الكل
```

## 🎓 تمرين سريع

جرب الآن:

1. **افتح** أي صورة
2. **اسحب** من الزاوية العلوية اليمنى إلى السفلية اليسرى
3. **لاحظ** القيم تتحدث تلقائياً
4. **عدّل** قيمة العرض بإضافة 100
5. **شاهد** المعاينة تتغير
6. **احفظ**

### استخدم السحب عندما:
- ✅ تريد السرعة

- ✅ لا تحتاج دقة عالية جداً

- ✅ تريد دقة عالية
- ✅ تعرف الأبعاد بالضبط
- ✅ تريد تكرار نفس القياسات

### استخدم الإجراءات السريعة عندما:
- ✅ تريد البداية السريعة
- ✅ المنطقة نصف الصورة أو المركز
- ✅ ستعدل بعدها
